# practice

A repo to practice coding challenges.

## Projects

### leetcode

Java solutions to LeetCode problems, organized by category:

Solution implementations are validated with unit tests. 

| Category | Problems |
| --- | --- |
| Arrays | Two Sum, Three Sum, Best Time to Buy/Sell Stock, Container With Most Water, Contains Duplicate, Jump Game, Maximum Subarray, Missing Number, Rotate Image, Spiral Matrix |
| Backtracking | Combination Sum |
| Binary Search | Find Min in Rotated Sorted Array, Search in Rotated Array |
| Bits | Number of One Bits, Reverse Bits |
| Design | LRU Cache |
| DP | Climbing Stairs, Counting Bits, Longest Increasing Sequence, Maximum Product Subarray, Word Break |
| Graphs | Clone Graph, Pacific Atlantic Water Flow, Unique Paths |
| Heap | Median Finder |
| Intervals | Insert Interval, Merge Intervals, Non-Overlapping Intervals |
| Linked List | Has Cycle, Merge Sorted Lists, Merge Two Sorted Lists, Remove Nth Node, Reorder List, Reverse Linked List |
| Stack | Valid Parentheses |
| Strings | Group Anagrams, Longest Palindromic Substring, Longest Repeating Character Replacement, Longest Substring Without Repeating, Palindromic Substrings, Ransom Note, Valid Anagram, Valid Palindrome |
| Trees | Invert Binary Tree, Maximum Depth Binary Tree, Same Tree, Serialize Binary Tree, Subtree of Tree |

## Tech Stack

- **Language:** Java 21
- **Build:** Gradle (Kotlin DSL)
- **Tests:** TestNG 7.5.1

## Running Tests

```bash
cd leetcode
./gradlew test
```
